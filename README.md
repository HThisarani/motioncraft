### Project Title

```md
# MotionCraft
```

---

## Project Overview

```md
MotionCraft is a frontend web application focused on motion-driven user interfaces and cinematic web experiences. The project demonstrates how modern animation techniques, background video, and clean layouts can be combined to create an immersive and engaging website.
```

---

## Features

```md
The application includes a fullscreen video-based hero section, animated navigation, scroll-triggered content sections, and interactive UI components designed with performance and responsiveness in mind.
```

---

## Technology Stack

```md
Frontend Framework: React (Vite)  
Styling Framework: Tailwind CSS  
Animation Techniques: CSS Animations, Intersection Observer API, requestAnimationFrame  
Media Assets: MP4 background video  
Version Control: Git and GitHub  
Deployment Platform: Vercel
```

## Getting Started (Local Setup)

### Step 1 – Clone the Repository

```bash
git clone https://github.com/HThisarani/motioncraft.git
cd motioncraft
```

---

### Step 2 – Install Dependencies

```bash
npm install
```

---

### Step 3 – Run the Development Server

```bash
npm run dev
```

---

### Application Access

```md
The application will be available at:

http://localhost:5173
```

---

## Deployment

```md
This project is deployed using Vercel.

Deployment Process:
1. Push the repository to GitHub
2. Import the repository into Vercel
3. Select Vite as the framework
4. Deploy the application
```

---

## Purpose

```md
The purpose of this project is to demonstrate modern frontend development practices, motion-based UI design, and the integration of video and animation in web applications. It serves as a portfolio and educational project showcasing advanced CSS animations, 3D transforms, and interactive user experiences.
```
